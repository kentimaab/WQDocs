# WideQuick Documentation

Official documentation for the **WideQuick Modular Framework** and its derived
concepts — **BMS** (Building Management System) and **WWT** (Water & Wastewater
Treatment) — developed by [Kentima AB](https://www.kentima.com), Staffanstorp, Sweden.

**Read the documentation:** [widequick.com/docs](https://widequick.com/docs)
&nbsp;·&nbsp; Swedish: [widequick.com/sv](https://www.widequick.com/sv/)

---

## What's documented

| Section | Scope |
| --- | --- |
| **WideQuick** | Core platform — installation, licensing, deployment, and the WideQuick Designer and Runtime. |
| **Modular Framework** | The shared engine behind WideQuick concept applications — modules (Alarms, Maintenance, Calendar, History, Reports, Dashboards, Maps, Documents, Logbook), the suffix system, multiview, database schemas, and release notes. |
| **BMS** | Building Management — configuration recipes, system views (ventilation, heating, electrical), customization, and operations. |
| **WWT** | Water & Wastewater Treatment — process visualization and control; mirrors the BMS structure with WWT-specific content. |
