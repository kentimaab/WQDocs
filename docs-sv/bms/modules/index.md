# Moduler

WideQuick BMS organiserar sina funktioner och sin funktionalitet i moduler. Varje modul täcker ett distinkt område av systemet. Core-modulen är grunden som varje installation bygger på. Den tillhandahåller navigering, larm, teman, användarhantering och grafiska symboler. Övriga moduler är byggda för att fungera tillsammans med Core och kan läggas till eller tas bort för att matcha projektets behov.
