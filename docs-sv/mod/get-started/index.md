# Kom igång med WideQuick MOD

<img src="../../assets/MOD-2.png" style="width:100%;box-shadow: 2px 2px 20px rgba(0, 0, 0, 0.5);"></img>

## Vad är WideQuick MOD

WideQuick MOD är Kentimas modulära ramverk för att bygga HMI- och SCADA-applikationer.
Det ger en strukturerad projektgrund med ett bibliotek av produktionsklara moduler
som täcker de funktioner varje system behöver — från larm och navigation till underhålls-
schemaläggning och rapportering. Integratörer inkluderar endast de moduler projektet
kräver och lämnar resten utanför.

För en fullständig förklaring av ramverksarkitekturen, moduloberoende och hur
MOD förhåller sig till BMS och WWT, se [WideQuick Modular Framework](../concepts/index.md).

## Vad ingår

Följande moduler ingår direkt:

* **Larm** — Larmhantering och -övervakning
* **Larmsändning** — E-post- och SMS-aviseringar för larmhändelser
* **Kalender** — Schemaläggning och tidsbaserad styrning
* **Dashboards** — Anpassningsbara översiktsdisplayer
* **Dokument** — Dokumentlagring och -hantering
* **Historik** — Historisk dataloggning och trender
* **Loggbok** — Driftsanteckningar
* **Underhåll** — Underhållsschemaläggning och -uppföljning
* **Kartor** — Interaktiva kartvyer med statusindikatorer
* **Navigation** — Plug-and-play-navigation som bygger sig själv, med stöd för
behörighetsbaserade adaptiva menyer
* **Popups** — Färdigbyggda objektpopups för processinteraktion
* **Rapporter** — Automatiserad rapportgenerering och schemaläggning
* **Grafiska symboler** — Omfattande och utbyggbara symbolbibliotek med förkonfigurerade
P&ID-symboler klara att använda direkt
* **Teman** — Anpassa applikationen för att passa dina kunder eller din egen grafiska
profil
