# Teman

WideQuick MOD använder ett temasystem byggt på Material Design 3 (M3). Varje färg i applikationen, inklusive bakgrunder, text, kanter, ytor och statusindikatorer, hämtas från det aktiva temat snarare än att vara ett fast värde. Att byta tema uppdaterar hela applikationen på en gång.

Teman levereras i ljusa och mörka varianter. Båda kan väljas i **WideQuick Designer®** och växlas under körning. Det aktiva valet sparas mellan omstarter.

Eftersom teman följer M3-standarden kan ett komplett tema med alla färgroller, höjdytor och mörka/ljusa varianter genereras från en enda varumärkesfärg. Det gör det praktiskt att bygga teman som matchar en kunds grafiska profil, och resultatet är sammanhängande, visuellt distinkt och standardenligt utan att manuellt definiera varje roll.
