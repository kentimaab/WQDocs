# Kärna

Kärna är grunden i varje WideQuick MOD-installation. Till skillnad från valfria moduler är Kärna alltid närvarande och den tillhandahåller navigationsstrukturen, larmhantering, temaSystem, användarbehörigheter och de grafiska symbolbibliotek som alla andra moduler bygger på.

## Innehåll { #contents }

### [Larm](alarms/index.md) { #alarms }

* [**Kom igång**](alarms/get-started.md) — Larmvyer, kvittering av larm och larmlistor i processvyer.
* [**Konfigurering**](alarms/configuring.md) — Skapa larmgrupper, lägga till larm och notifieringsscheman.
* [**Utöka**](alarms/extending.md) — E-post- och SMS-notifieringar.

---

### [Grafiska symboler](Graphical-symbols/index.md) { #graphical-symbols }

* [**Symbolbibliotek**](Graphical-symbols/Symbols.md) — Komponenter, spjäll, motorer, sensorer, ventiler med mera.
* [**Äldre symboler**](Graphical-symbols/legacy.md) — Symboler från tidigare generationen i WideQuick BMS 8.0.

---

### [Navigation](Navigation/index.md) { #navigation }

* [**Utöka**](Navigation/extending.md) — GoTo-funktion, anpassade navigationsikoner.
