# Kalender

Kalendermodulen ger en visuell översikt över händelser och kommande underhållsdeadlines. Den stöder tre vyer (månad, vecka och dag) och kan navigeras framåt och bakåt i tid.

Händelser hämtas från två källor: anpassade händelser som skapas direkt i kalendern, samt underhållsdeadlines som hämtas automatiskt från [Underhållsmodulen](../maintenance/index.md). En konfigurerbar påminnelse visas ett angivet antal dagar innan varje underhållsdeadline.
