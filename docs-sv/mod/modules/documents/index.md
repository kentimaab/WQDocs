# Dokument

Dokumentmodulen ger operatörer tillgång till filer direkt från HMI:n. Både lokala filer som lagras på servern och länkar till onlineresurser kan göras tillgängliga. Dokument kan även kopplas till enskilda objekt, vilket gör att relevanta manualer, certifikat eller datablad finns tillgängliga direkt från [objektets popup](../../reference/Popup/Documents.md).
