# Moduler

WideQuick MOD organiserar sina funktioner i moduler. Varje modul täcker ett eget område i systemet. Core-modulen är grunden som varje installation bygger på. Den tillhandahåller navigation, larm, teman, användarhantering och grafiska symboler. Övriga moduler är byggda för att fungera tillsammans med Core och kan läggas till eller tas bort för att matcha projektets behov.
