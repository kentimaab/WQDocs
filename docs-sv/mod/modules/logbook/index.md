# Loggbok

Loggboksmodulen ger ett sätt att registrera fritextanteckningar kopplade till specifika delar av systemet. Poster organiseras efter ämne (en hierarkisk sökväg som återspeglar var i systemet anteckningen hör hemma) och kan valfritt grupperas i namngivna kontexter.

Loggboken är tillgänglig på tre ställen: globalt under **Dokument & Loggbok → Loggbok**, i fliken [**Loggbok**](../../reference/Popup/Logbook.md) i valfritt objektpopup, samt via menyn **SpeedDial** i processvyer.
