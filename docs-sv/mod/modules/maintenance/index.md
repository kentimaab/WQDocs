# Underhåll

Underhållsmodulen låter användare schemalägga och följa upp underhållsuppgifter för objekt i systemet. Uppgifter kan skapas manuellt för valfritt objekt eller genereras automatiskt enligt ett återkommande schema. Mallar påskyndar skapandet av återkommande uppgiftstyper genom att fördefiniera typ, intervall och beskrivning.

Det finns två sätt att komma åt underhållsuppgifter. Vyn **Underhåll - Lista** i huvudmenyn visar alla uppgifter för samtliga objekt i projektet. Fliken [**Underhåll**](../../reference/Popup/Maintenance.md) i objektpopupen visar bara uppgifterna för det specifika objektet.
