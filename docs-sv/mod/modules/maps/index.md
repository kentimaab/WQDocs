# Kartor och indikatorer

WideQuick Modular Framework inkluderar en inbyggd modul för Kartor och indikatorer som används för att bygga interaktiva kartsvyer. Den tillhandahåller en uppsättning indikatorer för att markera platser, visa live-status, gruppera nålar, visualisera rörnätverk och filtrera larm baserat på vad som för tillfället är synligt på kartan.
