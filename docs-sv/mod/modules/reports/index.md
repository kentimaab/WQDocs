# Rapporter
WideQuick Modular Framework inkluderar en inbyggd rapportmodul som låter dig
generera, schemalägga och anpassa rapporter baserade på larm- och energidata. Det här avsnittet
täcker allt från att använda standardmallarna till att skapa helt egna mallar.
