# Dokument

Popup-fönstret Dokument visar alla dokument som är länkade till det valda objektet. Dokument
kan förhandsgranskas direkt i popup-fönstret och nya dokument kan länkas från
datorn. Det är alltid synligt i flikmenyn oavsett vilka suffix objektet har. För mer information om hur dokumentsystemet fungerar, se
[Dokument](../../modules/documents/index.md).

![Documents popup](/Images/Popups/Documents.png)

## Länkade dokument

Den vänstra panelen listar alla dokument som är länkade till det valda objektet, grupperade under
**Objekt**. Att välja ett dokument i listan visar en förhandsgranskning av det i
den högra panelen. Den inbyggda dokumentvisaren stöder navigering mellan sidor, zoomkontroller och automatisk zoom.

## Åtgärder

Följande åtgärder är tillgängliga längst ned i popup-fönstret:

* **Förhandsgranska** — öppnar det valda dokumentet i förhandsgranskningspanelen
* **Ta bort dokumentlänk** — tar bort länken mellan det valda dokumentet och
objektet. Dokumentet tas inte bort från systemet.
* **Redigera dokumentlänkar** — öppnar en dialog för att lägga till eller hantera dokumentlänkar för
objektet, vilket gör det möjligt att länka dokument från datorn
