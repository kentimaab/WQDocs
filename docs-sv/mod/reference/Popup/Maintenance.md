# Underhåll

Underhållspopupen visar alla underhållsuppgifter som är schemalagda för det valda
objektet. Den är alltid synlig i flikmenyn oavsett vilka suffix objektet har. För
mer information om hur underhållssystemet fungerar, se
[Underhåll](../../modules/maintenance/index.md).

![Maintenance popup](/Images/Popups/Maintenance.png)

## Underhållsinformation

Listan **Underhållsinformation** visar alla underhållsuppgifter som är kopplade till
objektet. Varje post visar följande information:

* **Uppgift** — underhållsuppgiftens namn
* **Prioritet** — uppgiftens prioritetsnivå
* **Tilldelad tekniker** — den person som ansvarar för uppgiften
* **Deadline** — det datum då uppgiften måste vara slutförd
* **Status** — uppgiftens aktuella status, till exempel Planerad eller Pågående
* **Återkommande** — om uppgiften upprepas enligt ett schema

## Åtgärder

Följande åtgärder är tillgängliga längst ned i popupen:

* **Ny underhållsuppgift** — skapar en ny underhållsuppgift för detta objekt
* **Redigera underhåll** — redigerar den valda underhållsuppgiften
* **Ta bort underhåll** — tar bort den valda underhållsuppgiften
