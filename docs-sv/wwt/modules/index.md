# Moduler

WideQuick WWT organiserar sina funktioner i moduler. Varje modul täcker ett specifikt område av systemet. Core-modulen är grunden som varje installation bygger på. Den tillhandahåller navigering, larm, teman, användarhantering och grafiska symboler. Övriga moduler är byggda för att fungera tillsammans med Core och kan läggas till eller tas bort beroende på vad projektet behöver.
