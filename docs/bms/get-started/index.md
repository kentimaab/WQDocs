# Get Started with WideQuick BMS

## What is WideQuick BMS?

WideQuick BMS is a concept application for Building Management System built on top of
WideQuick Modular Framework. It extends the framework with a set of modules
specifically designed for building automation. The concept covers everything from alarm
management and energy monitoring to maintenance scheduling and document handling.

WideQuick BMS is designed for integrators building and commissioning BMS projects,
as well as facility managers and building owners who need a reliable and flexible
system for monitoring and controlling their building systems. It is suitable for a
wide range of applications including HVAC, heating, ventilation, and energy
management.

The following modules are included out of the box:

* **Alarms** — Alarm management and monitoring
* **Calendar** — Scheduling and time-based control
* **Dashboards** — Customizable overview displays
* **Documents** — Document storage and management
* **History** — Historical data logging and trends
* **Logbook** — Operational notes
* **Maintenance** — Maintenance scheduling and tracking
* **Maps** — Interactive map views with status indicators
* **Reports** — Automated report generation and scheduling
* **Audit Trail** — Gain oversight of important variable changes and who did it
* **Extensive and extendable symbol libraries** — Visual components ready to use, save time recreating symbols.
* **Theming** — Adapt or whitelist the application to suit your customers or your own graphical profile.
* **Backup of parameters** — Save crucial parameters to be restored at a later point in time.
* **Automatic navigation** — Plug and play navigation that builds itself. Includes features for adaptive navigation based on user privilegies.
