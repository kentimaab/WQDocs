# Troubleshooting

Common issues and solutions for MOD modules.

* [Calendar](calendar.md) — Events not appearing, blank calendar, stale events.
* [Dashboards](dashboards.md) — Widget header undefined, blank signal values, missing alarms.
* [History](history.md) — Empty signal tree, missing signals, blank chart.
* [Logbook](logbook.md) — Missing entries, empty topic tree, wrong scope.
* [Maintenance](maintenance.md) — Widget showing zero, recurring task not created, legacy Ongoing status.
* [Reports](Reports.md) — No data available, no data appearing, Reports not sent
