# Graphical Symbols

WideQuick includes two types of symbol libraries for building process views —
basic symbols and wired up symbols. Together they cover the most common components
found in building automation and process control applications.

For information on the new two-layer animation system and why it improves on older
symbols, see [Why the new symbols](Symbols.md#why-the-new-symbols).

---

### [Symbols](Symbols.md)
* [**COMPONENTS**](Symbols.md#components) — Base symbols for building
custom process objects, including motors, pumps, valves, sensors and more.

---

### Wired up symbols
* [**DAMPERS**](Symbols.md#dampers) — Pre-configured digital and analog
damper objects in four orientations.
* [**MOTORS**](Symbols.md#motors) — Pre-configured motor objects for
compressors, fans, pumps, frequency converters, and revolving heat exchangers.
* [**SENSORS**](Symbols.md#sensors) — Pre-configured sensor objects for
standard, dual, differential, digital, and outdoor sensors.
* [**VALVES**](Symbols.md#valves) — Pre-configured two-way and three-way
valve objects in four orientations.
* [**OTHERS**](Symbols.md#others) — Pre-configured objects for energy
meters, flow meters, service couplers, timers, pushbuttons, and expansion vessels.

---

### Legacy symbols
* [**Legacy symbols**](legacy.md) — Earlier generation
symbols from WideQuick BMS 8.0. Automatically upgraded when importing old views.
Migration to the new symbols is recommended for new and updated projects.
