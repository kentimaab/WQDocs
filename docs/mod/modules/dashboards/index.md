# Dashboards

A Dashboard is a configurable view for monitoring signals, values and status
from different parts of the system in one place. Instead of navigating between
operating image, a dashboard lets the user bring the data that matters most into
a single view. The view can contain alarms, live values, histories, energy KPIs
and maintenance status.
