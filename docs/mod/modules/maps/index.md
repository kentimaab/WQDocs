# Maps & Indicators

WideQuick Modular Framework includes a built-in Maps & Indicators module for building
interactive map views. It provides a set of indicators for marking locations, displaying
live status, grouping pins, visualising pipe networks, and filtering alarms based on
what is currently visible on the map.
