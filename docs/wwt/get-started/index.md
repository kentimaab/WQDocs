# WideQuick Water and Wastewater Treatment (WWT)

<img src="/docs/Images/wwt/WWT-1_1000x246.png" style="width:100%;box-shadow: 2px 2px 20px rgba(0, 0, 0, 0.5);"></img>

WideQuick WWT is a concept application built on the [WideQuick Modular Framework](../../mod/index.md), purpose-built for water and wastewater treatment. Like [WideQuick BMS](../../bms/index.md), it inherits the full MOD foundation — modules, scripting, and tooling — extended with industry-specific functionality for water facilities.

## Extensive Symbol & Object Libraries

With the introduction of the framework, several symbol and object libraries, along with associated script libraries, are included. The symbol libraries contain over a hundred unique P&ID symbols based on ISO and ISA standards. These symbols are pre-configured to display colors and provide operators with information about the status of the object the symbol represents.

The accompanying object libraries include components such as buttons and value displays. The value displays can be connected to objects in work views to visualize values directly in process images. These displays come with built-in functionality for handling units and decimals.

WideQuick Modular Framework makes it easy for users to add or remove modules as needed to tailor the application for specific purposes. The flexible and modular structure also simplifies the updating of specific parts of the application in the future.
